# Пользовательские сценарии — Ева

Источник персон для протокола «Глазами пользователя».
Лид копирует релевантные сценарии в контракт каждой user-facing задачи.

## Персоны

| Персона | Кто | Контекст | Типичное сообщение |
|---------|-----|----------|-------------------|
| Маша | 25 лет, первый раз | 0 сообщений, профиль пуст, ничего не знаем | "Привет" |
| Аня | 28 лет, активная | 30 сообщений, профиль заполнен, есть цель и паттерны | "Мне грустно" |
| Лена | 35 лет, вернулась | Не писала 5 дней, есть история | "Я всё провалила" |

## Чеклисты по персонам

### Маша (новая)
- [ ] Все секции имеют fallback (профиль, процедурная память, эпизоды)
- [ ] Нет пустых блоков, "undefined" или обрезанного текста в результате
- [ ] Тон тёплый, приглашающий, не перегруженный
- [ ] Ева знает что данных нет и не додумывает

### Аня (активная)
- [ ] Все данные попали в результат (профиль, эпизоды, паттерны, цель)
- [ ] Ничего не потерялось при сборке/обрезке
- [ ] Ева помнит конкретику (имена, события, инсайты)
- [ ] Релевантные эпизоды подобраны по текущему сообщению

### Лена (вернувшаяся)
- [ ] Пауза учтена и отражена в контексте
- [ ] Нет "бодрого приветствия" после 5 дней молчания
- [ ] Ева помнит где остановились
- [ ] Тон: мягкое "рада что вернулась", не допрос "где была"
