# 6. МАКЕТЫ ЭКРАНОВ — Мини-приложение Евы

## Принцип
ASCII-макеты всех экранов. Код должен воспроизводить эти макеты. Стиль: Apple Health / iOS Settings. Палитра: розовый акцент (#FF6B8A), белый фон (#FFFFFF), серый фон секций (#F2F2F7). Контраст текста на акценте: #D4466A (WCAG AA).

Тёмная тема: инвертирован фон (#1C1C1E / #2C2C2E), текст (#FFFFFF / #EBEBF5), акцент тот же (#FF6B8A).

---

## 6.1. Навигация (Tab Bar)

Всегда внизу экрана. 3 вкладки.

```
┌──────────────────────────────────┐
│                                  │
│         (содержимое экрана)      │
│                                  │
├──────────────────────────────────┤
│  ☀️ Сегодня    📈 Мой путь    👤 Профиль  │
│  ─────────                       │
│  (розовый)    (серый)    (серый)  │
└──────────────────────────────────┘
```

| Элемент | Действие при нажатии |
|---------|---------------------|
| Сегодня | Переключить на HomePage |
| Мой путь | Переключить на ProgressPage |
| Профиль | Переключить на ProfilePage |

Активная вкладка: розовый (#FF6B8A) иконка + подчёркивание. Неактивные: серый.

## 6.3. Экран «Сегодня» — активный пользователь (есть цели)

Фаза: ЦЕЛЬ или РИТМ, 20+ сессий, есть цель и обязательства.

```
┌──────────────────────────────────┐
│                                  │
│  Привет, Маша!                   │
│  🔥 Серия: 5 дней               │
│                                  │
│  ── Мысль на сегодня ──────────  │
│  ┌────────────────────────────┐  │
│  │  ✨ «Ты не тряпка. Ты      │  │
│  │  учишься выбирать себя»    │  │
│  └────────────────────────────┘  │
│                                  │
│  ── Задание ───────────────────  │
│  ┌────────────────────────────┐  │
│  │  🎯 Отказать следующему    │  │
│  │     клиенту со скидкой     │  │
│  │     До: пятница            │  │
│  │                            │  │
│  │  [ Выполнено ✓ ]           │  │
│  └────────────────────────────┘  │
│                                  │
│  ┌────────────────────────────┐  │
│  │    [ Написать Еве → ]      │  │
│  │    Откроется чат            │  │
│  └────────────────────────────┘  │
│                                  │
├──────────────────────────────────┤
│  ☀️ Сегодня    📈 Мой путь    👤 │
└──────────────────────────────────┘
```

| Элемент | Действие |
|---------|----------|
| Серия дней | Статичный текст. Отображается если streak > 1 |
| «Мысль на сегодня» | Аффирмация, сгенерированная GPT-4o-mini с контекстом из профиля |
| Задание | Текущее обязательство из `commitments`. Если несколько — показать одно, остальные по свайпу |
| «Выполнено ✓» | Haptic feedback + пометить как выполненное (PUT /api/user/commitment/{id}/done) + показать следующее |
| «Все задания выполнены!» | Показать когда commitments пуст. Иконка 🎉 |
| «Задание появится после разговора» | Показать если commitments никогда не было |
| «Написать Еве →» | Открыть чат с Евой |

```
┌──────────────────────────────────┐
│                                  │
│         [  Аватар 96px  ]        │

│     Разговоров: 23 · Базовый     │

│  ┌────────────────────────────┐  │
│  │  ⏰ Утреннее сообщение     │  │
│  │     09:00              [>] │  │
│  ├────────────────────────────┤  │
│  │  🔔 Напоминания            │  │
│  │     Вкл               [>] │  │
│  └────────────────────────────┘  │
│                                  │
│  ── Данные ────────────────────  │
│  ┌────────────────────────────┐  │
│  │  🗑️ Забудь тему...    [>] │  │
│  ├────────────────────────────┤  │
│  │  ⚠️ Удалить аккаунт   [>] │  │
│  └────────────────────────────┘  │
│                                  │
│  ── Ещё ───────────────────────  │
│  ┌────────────────────────────┐  │
│  │  ❓ Помощь             [>] │  │
│  ├────────────────────────────┤  │
│  │  ⚖️ Дисклеймер         [>] │  │
│  └────────────────────────────┘  │
│                                  │
│  v0.1.0 · Ева не заменяет       │
│  психолога                       │
│                                  │
├──────────────────────────────────┤
│  ☀️ Сегодня    📈 Мой путь    👤 │
└──────────────────────────────────┘
```

| Элемент | Действие |
|---------|----------|
| Аватар | Из Telegram (если есть) или первая буква имени |
| Имя | Из Telegram `first_name` |
| «Разговоров: N» | Статичный текст, sessions_count |
| Утреннее сообщение | Открыть popup: выбор времени (08:00 / 09:00 / 10:00 / Выкл). PUT /api/user/morning_time |
| Напоминания | Toggle: Вкл/Выкл. PUT /api/user/reminders |
| «Забудь тему...» | Открыть чат с Евой + deep link команда /forget. Ева спросит какую тему забыть |
| «Удалить аккаунт» | Popup подтверждения: «Все данные будут удалены. Это нельзя отменить.» → DELETE /api/user → показать «Готово, все данные удалены» |
| Помощь | Открыть чат с Евой |
| Дисклеймер | Popup: «Ева — AI-подруга. Она не заменяет психолога, психотерапевта или врача. Если тебе нужна профессиональная помощь — обратись к специалисту.» |

## 6.6. Убрано из MVP

Следующие элементы **НЕ показываются** в MVP:

| Элемент | Почему | Когда вернуть |
|---------|--------|---------------|
| Баннер подписки | MVP бесплатный для 10 тестеров | После валидации гипотезы |
| Режим общения (Коуч/Подруга/Астролог) | MVP = только «Тёплая подруга» | После валидации спроса |
| Стиль коучинга (Мягкий/Сбалансированный/Жёсткий) | Ева сама адаптирует стиль через процедурную память | Если тестеры попросят |
| Карта дня / Таро | Заменено на «Мысль на сегодня» (аффирмация) | Если режим «Астролог» подтвердится |
| История платежей | Нет платежей в MVP | При монетизации |
| Вечерний чек-ин | Не приоритет для MVP | После D7 если retention ок |

## 6.7. Тёмная тема

Тёмная тема определяется через `Telegram.WebApp.colorScheme` и применяется автоматически.

### Цветовая схема

| Элемент | Светлая | Тёмная |
|---------|---------|--------|
| Фон страницы | #F2F2F7 | #1C1C1E |
| Фон карточки/секции | #FFFFFF | #2C2C2E |
| Основной текст | #000000 | #FFFFFF |
| Вторичный текст | #8E8E93 | #EBEBF599 |
| Акцент (кнопки, активная вкладка) | #FF6B8A | #FF6B8A |
| Текст на акценте | #FFFFFF | #FFFFFF |
| Разделитель | #C6C6C8 | #38383A |
| Tab bar фон | #FFFFFF | #1C1C1E |
| Прогресс-бар фон | #E5E5EA | #38383A |
| Прогресс-бар заполнение | #FF6B8A | #FF6B8A |

### Пример — «Сегодня» в тёмной теме

```
┌──────────────────────────────────┐
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ (#1C1C1E)    │
│                                  │
│  Привет, Маша!        (белый)    │
│  🔥 Серия: 5 дней    (серый)    │
│                                  │
│  ┌────────────────────────────┐  │
│  │  (#2C2C2E)                 │  │
│  │  ✨ «Ты учишься выбирать  │  │
│  │  себя»           (белый)   │  │
│  └────────────────────────────┘  │
│                                  │
│  ┌────────────────────────────┐  │
│  │  (#2C2C2E)                 │  │
│  │  🎯 Отказать клиенту      │  │
│  │  [ Выполнено ✓ ] (#FF6B8A) │  │
│  └────────────────────────────┘  │
│                                  │
│  [  Написать Еве →  ] (#FF6B8A) │
│                                  │
├──────────────────────────────────┤

│  ☀️ Сегодня(розов) 📈(серый) 👤  │
└──────────────────────────────────┘
```

### Реализация

CSS-переменные в `:root` и `[data-theme="dark"]`:
```css
:root {
  --bg-page: #F2F2F7;
  --bg-card: #FFFFFF;
  --text: #000000;
  --text-secondary: #8E8E93;
  --accent: #FF6B8A;
  --separator: #C6C6C8;
}

[data-theme="dark"] {
  --bg-page: #1C1C1E;
  --bg-card: #2C2C2E;
  --text: #FFFFFF;
  --text-secondary: rgba(235, 235, 245, 0.6);
  --accent: #FF6B8A;
  --separator: #38383A;
}
```

Тема устанавливается при загрузке: `document.documentElement.dataset.theme = Telegram.WebApp.colorScheme`.

## 6.8. Состояния экранов

Каждый экран имеет 3 состояния:

**Loading (скелетон):**
```
┌────────────────────────────────┐
│  ████████████           (40%) │
│  ██████████████████     (60%) │
│                                │
│  ┌────────────────────────────┐│
│  │ ██████████████████████████ ││
│  │ ██████████████████████████ ││
│  └────────────────────────────┘│
└────────────────────────────────┘
```
Анимированные серые блоки с пульсацией. Структура повторяет реальный контент.

**Error:**
```
┌────────────────────────────────┐
│                                │
│            😔                  │
│  Не удалось загрузить данные   │
│                                │
│       [ Повторить ]            │
│                                │
└────────────────────────────────┘
```

**Empty (нет данных):**
```
┌────────────────────────────────┐
│                                │
│            ✨                  │
│  Появится после разговора      │
│  с Евой                        │
│                                │
└────────────────────────────────┘
```
