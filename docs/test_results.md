## Подготовка по сценариям

### Блок 1: Знакомство (сценарии 1-4)
- `/delete_account` перед каждым сценарием
- Для 3: нужно 2 предварительных сообщения (представиться Леной)
- Для 4: нужно 4 предварительных (представиться Аней, отвечать коротко)

### Блок 2: Зеркало (сценарии 5-7)
- Нужна заполненная память (10+ сообщений)
- Сценарий 5: рассказать про фриланс, маму, «не могу отказать» 2 раза
- Сценарий 6: рассказать про одиночество, работу
- Сценарий 7: рассказать про развод, страх бизнеса

### Блок 3: Кризис (сценарии 8-10) — ПРИОРИТЕТ
- Сценарий 8: чистый пользователь ОК
- Сценарий 9: нужен контекст (Ира, парень Дима, крики)
- Сценарий 10: нужен весёлый контекст

### Блок 4: Возвращение (сценарии 11-13)
- Нужна пауза между сообщениями (или мок в БД)
- Сценарий 11: контекст про маму, пауза 2 дня
- Сценарий 12: контекст про ИП, пауза 10 дней
- Сценарий 13: контекст про одиночество, пауза 5 дней

### Блок 5: Цели (сценарии 14-16)
- Сценарий 14: нужен контекст (50+ сообщений или мок фазы ЦЕЛЬ)
- Сценарий 15: нужна цель + обязательство + дедлайн
- Сценарий 16: нужна цель + контекст «Олег»

### Блок 6: Память (сценарии 17-18)
- Сценарий 17: нужен профиль с дочкой Алисой, сад
- Сценарий 18: нужен контекст без конкретного имени коллеги

### Блок 7: Edge-case (сценарии 19-20)
- Сценарий 19: любой контекст
- Сценарий 20: нужно записать голосовое > 3 мин
